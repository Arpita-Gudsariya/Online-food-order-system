# 🍔 Online Food Ordering System

## 📌 Project Description

This is a full-stack Online Food Ordering System built using **Node.js, Express, and React**.
Users can browse food items, place orders, and manage data efficiently.

---

## 🚀 Features

* 🛒 Browse food items
* 📦 Place orders
* 👤 User management
* 🔗 Backend API integration
* ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Version Control: Git & GitHub

---

## ⚙️ Installation & Setup

1️⃣ Clone the repository:

```
git clone https://github.com/Arpita-Gudsariya/Online-food-order-system.git
```

2️⃣ Go to project folder:

```
cd Online-food-order-system
```

3️⃣ Install dependencies:

👉 Backend:

```
cd backend
npm install
```

👉 Frontend:

```
cd frontend
npm install
```

4️⃣ Run the project:

👉 Backend:

```
npm start
```

👉 Frontend:

```
npm start
```
